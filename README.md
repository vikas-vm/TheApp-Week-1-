## Installation

- `python/python3 -m venv env`
- `source activate env`
- `git clone https://github.com/vikas-vm/TheApp-Week-1-.git`
- `git checkout vi-dev`
- `pip install -r requirements.txt`
- `python manage.py runserver`

## Built With

- Python
- Django
- DjangoRestFramework
- JWTAuthentication
- DjangoRestFrameworkSimpleJWT
- Django_Filter
